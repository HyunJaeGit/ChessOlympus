# 🏛️ ChessOlympus: Hades VS Zeus

### **"체스올림푸스 : 하데스 vs 제우스"**

---

## 📖 게임 소개

**ChessOlympus**는 그리스 신화를 배경으로 한 **RPG형 턴제 전략 게임**입니다.
각기 다른 능력과 스킬을 가진 신들을 지휘해 승리를 쟁취하세요.

## 📜 게임 규칙 (Core Rules)

1. **승리 조건**:
* 상대 팀의 **리더**를 처치하면 즉시 승리합니다.
* 인공지능(AI) 상대와 솔로 플레이 가능


2. **턴제 전투**:
* 자신의 턴에 유닛을 선택하여 **이동**할 수 있습니다.
* 모든 이동이 끝나면 **자동공격**과 함께 상대방에게 턴이 넘어갑니다.


3. **유닛 상성**:
* 각 유닛은 고유의 **공격력, ,반격력, 체력, 사거리, 이동력**을 가집니다.
* 유닛마다 특수한 **스탯과 스킬**을 보유하고 있어 전략적 활용이 중요합니다.


4. **조작 방법**:
* **마우스 클릭**: 유닛 선택 및 타일 이동 / 공격 대상 지정
